# Delphoi

「出る」という言葉にフォイフォイ反応してしまうbot

## Usage & Installation

`hg cl` して `shly delphoi` するだけ

### Requirements

* clozure cl
* quicklisp
* shelly
* delphoi を quicklisp のパスが通ったところに置ける程度の能力
